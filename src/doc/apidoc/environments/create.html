POST /api/organizations/:organization_id/environments
Create an environment in an organization

Examples

POST /api/organizations/org_2d4c613/environments/
{
  "environment": {
    "name": "env 2d4c613",
    "prior": 72
  }
}
200
{
  "name": "env 2d4c613",
  "prior": "Library",
  "id": 73,
  "created_at": "2012-06-05T06:56:07Z",
  "organization": "org_2d4c613",
  "description": null,
  "updated_at": "2012-06-05T06:56:07Z",
  "organization_id": 28,
  "library": false,
  "prior_id": 72
}

Params

Param name Description
environment
optional
Value: Hash
environment[name]
optional
environment[prior]
optional

id of an environment that is prior the new environment in the chain, it has to be either library or an envrionment at the end of the chain
