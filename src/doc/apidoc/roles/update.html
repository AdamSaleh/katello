PUT /api/roles/:id
Update a role

Examples

PUT /api/roles/32
{
  "role": {
    "name": "role_",
    "description": "new description of the role"
  }
}
200
{
  "updated_at": "2012-06-05T07:05:03Z",
  "name": "role_",
  "id": 32,
  "description": "new description of the role",
  "locked": false,
  "created_at": "2012-06-05T07:05:03Z"
}

Params

Param name Description
role
optional
